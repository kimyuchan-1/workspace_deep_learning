# 파이토치 2.9
## 활성화 함수: 신경망, 뉴런, "비선형성"을 부여, 복잡한 패턴을 학습하는데 도움

- 1세대(고전): sigmoid, tanh 
    => 속도가 좀 느림
    => 기울기 소실 문제
- 2세대: ReLU
    => 기울기 소실 문제 해결(나름의)
- 3세대(확률): GELU(트랜스포머)
    => 언어 모델의 표준
    => 입력에 따른 가중치를 부여
- 4세대(진화): Mish(Swish 개선판)

---

tanh: RNN/LSTM에 주로 사용
ReLU: CNN에 주로 사용
GELU: 트랜스포머

---

이진 분류의 출력층 : sigmoid
다중 분류의 출력층 : softmax

## 손실 함수: 예측값(y_hat)과 실제값(레이블)간의 차이를 측정

- 1세대: 고전
    => MSE
- 2세대: 기계 학습
    => Cross Entropy
- 3세대: 딥러닝
    => Focal
    => Label Smoothing
    => Triplet Loss

---

다중 클래스 분류: Cross Entropy
이진 분류: BCEwithLogitsLoss
멀티 라벨 분류: BCEwithLogitsLoss
회귀: MSELoss, L1Loss

## 최적화 함수: 손실 함수의 기울기를 이용해서 모델의 가중치를 업데이트하는 것

- 1세대: 고전
    => SGD(1951)
- 2세대: (파라미터별) 적응형
    => RMSprop: 학습률 감소 문제를 완만
- 3세대: (현대) 적응형
    => Adam
- 4세대: (대규모) 적응형
    => LAMB

---

SGD: 모든 작업에 다 어울림. 단, 튜닝이 너무 힘듦
Adam(AdamW): 모든 작업에 다 어울림. 그리고 튜닝할만 함
RMSprop: RNN(구형), 튜닝이 애매함
